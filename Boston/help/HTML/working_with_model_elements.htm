Working with existing Model Elements

 Top  Previous  Next

Model Elements are those conceptual entities and associated text that provide the structure of an ORM Diagram and a Model.

e.g. Entity Types, Value Types, Fact Types, Role Constraints and Model Notes are all Model Elements.

 

In Boston, any shape or text that can be viewed within a Page is a Model Element. Model Elements are, in effect, the 'elements' that go into the construction of a Model.

 

Boston provides a convenient way to view the properties of any Model Element within a Page of a Model by viewing those properties within the Properties Toolbox (see The Properties Toolbox ).

 

The help topic, Viewing a Model Element's Properties, outlines how to view the properties of a Model Element within the Properties Toolbox.
