Selecting Model Elements

 Top  Previous  Next

The following steps outline how to selected a Model Element within a Page:

 

NB The steps in this help topic apply to a Page that already has Model Elements on the Page. For steps on how to add Model Elements to a Page, see the help topic: Adding Model Elements to a Model

 

1.Make sure that a Page is open for editing within the Boston workbench;
2.With your mouse, maneuver the mouse cursor over the Model Element that you would like to select;
3.Left-click on your mouse.
The Model Element that you have just selected will now have a blue outline.
