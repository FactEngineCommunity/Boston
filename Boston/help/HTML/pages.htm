Each Page of an ORM Model is a view over an ORM Model. Pages are displayed within the Model Explorer as a list of Pages for a Model.

 

For a detailed description of Pages, see the help topic, Managing Pages

 

Pages are displayed in the Model Explorer as document icons under the Model icon within the tree displayed within the Model Explorer. The picture below shows a set of three Pages for a Model called, 'My Model'.

 


 

Pages within a Model within the Model Explorer

 

To create an ORM Model in Boston you need at least one Page view over that Model.

 

This chapter assumes that you have already created a Page for a Model and opened that Page to start creating an ORM Diagram.
