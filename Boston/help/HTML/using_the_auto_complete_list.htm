Using the Auto-Complete Tool

 Top  Previous  Next

When typing VAQL sentences into the Virtual Analyst toolbox the Auto-Complete tool/list will periodically appear under the last character you have typed.

 

The Auto-Complete tool is designed to significantly accelerate the process of typing VAQL sentences, and provides a list of the next words (or 'tokens') to complete the sentence.

 

In the example below, the Auto-Complete tool shows a list of Data Types that are valid, to complete the "IS WRITTEN AS" sentence.
