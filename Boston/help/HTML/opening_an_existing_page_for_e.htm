Opening an existing Page for editing

 Top  Previous  Next

The following steps outline how to open an existing Page for editing from within the Model Explorer:

 

1.Make sure the Model Explorer is open within the Boston workbench;
2.Navigate to a Model and Page within the Model Explorer;
3.Right-click on the Page within the Model that you wish to open for viewing or editing within the Boston workbench.
A context-menu will appear with the options that you can select for that Page.
 

 
4.Click on the [Edit Page] option within the context-menu;
The Page will be opened within the Boston workbench for viewing and editing.
