Data Type Not Specified Error

 

Each Value Type within a Model requires a specified Data Type. When this Model Error appears in the Error List Toolbox, the name of the Value Type to which the error relates will be displayed within the error message.

 

Error Resolution

 

To resolve this error, set the desired Data Type of the respective Value Type (i.e. from <Data Type Not Set>)
