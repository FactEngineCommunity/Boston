Emptying an ORM Model

 Top  Previous  Next

The following steps outline how to Empty an existing ORM Model of all its Model Elements.

 

1.Make sure the Model Explorer is open within the Boston workbench
2.Right-click on the Model that you would like to Empty, within the list of Models within the Model Explorer.
You will be provided with the Model context menu.
 

 
3.Select the [Empty Model] menu option within the Model context menu;
4.The Model will be emptied of all Model Elements, and will (in effect) be an empty Model.
