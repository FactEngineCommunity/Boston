Adding Value Constraints to a Value Type

 

Value Constraints

 

Value Constraints have three main types:

 

1.Enumerations
2.Value Ranges
3.Multiple Combinations of Enumerations and/or Value Ranges

 

Enumerations

 

An Enumerated Value Constraint lists the discrete Values that are valid for a Value Type.

 

The picture below shows an Enumerated Value Constraint for the Value Type, 'SexCode', which in this instance forms the Reference Mode for the Entity Type, 'Sex'.

The discrete values are 'M' for Male and 'F' for Female.

Value Ranges

 

A Value Constraint may constrain a Value Type to a range of values.

 

The picture below shows a Value Constraint Range for the Value Type, 'LowercaseChar', limiting the values to the letters 'a' through 'z'.

Multiple Value Constraints

 

A Value Type may have multiple Value Constraints.

 

The picture below shows a Value Type, 'WheelCount' (e.g. the number of wheels on a vehicle), with values limited in the ranges, 2..4 and 16..18
