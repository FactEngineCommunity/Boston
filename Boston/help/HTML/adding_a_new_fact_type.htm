Using the keyboard to add a new Fact Type to a Model

 

The quickest and easiest way to add a new Fact Type to a Model is to use the keyboard.

 

The following steps outline how to add a new Fact Type to a Model using the keyboard:

 

Adding a Unary Fact Type to a Model

 

1.Select the Object Type to which you would like to add a Unary Fact Type;
 

 
2.Press the [R] key on your keyboard.
A new Fact Type with a single Role will be added to the selected Object Type
