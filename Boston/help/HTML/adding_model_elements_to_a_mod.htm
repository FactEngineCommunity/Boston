Adding Model Elements to a Model

 Top  Previous  Next

This chapter outlines how to add Model Elements to a Model.

 

There are many ways to add a Model Element to a Page and Model:

 

a) Using the Toolbox;

b) Using Shortcut Key combinations (Shortcut Key Combinations);

c) Copying and Pasting Model Elements from another Page or Model (Copying and Pasting Model Elements, Adding a Model Element to one Model from another Model)
