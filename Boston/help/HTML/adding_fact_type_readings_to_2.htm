Adding Fact Type Readings to a Fact Type

 Top  Previous  Next

The following steps outline how to add a Fact Type Reading to a Fact Type.

 

1.Make sure the Fact Type Reading Editor is open within the Boston workbench;
2.Select the Fact Type to which you would like to add a Fact Type Reading, within the Page;
The name of the Fact Type will appear at the top of the Fact Type Reading Editor.
3.Type the Fact Type Reading in the textbox at the top of the Fact Type Reading Editor

 
 
               
 

4.Press [Enter];
The Fact Type Reading for the Fact Type will be added to the list of Fact Type Readings for the Fact Type within the grid at the base of the Fact Type Reading Editor.
