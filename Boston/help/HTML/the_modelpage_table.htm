The ModelPage table

 Top  Previous  Next

The ModelPage table stores details about each 'Page' within a Model.

 

Field

Description

PageId

The unique identifier of the Page.

PageName

The name of the Page.

ModelId

The unique identifier of the Model to which the Page belongs.

LanguageId

Foreign Key Reference to the ModelLanguage table. Represents the Language that should be used to interpret Models stored within the MetaModel.

 

NB This version of Boston only supports the interpretation of ORM Models from the MetaModel.

IsCoreModelPage

Not used in this version of Boston. Provided for consistency with the Richmond product by Viev Pty Ltd.

ShowFacts

Set to True if the Fact Table of Facts is to be displayed to the user on loading of the Page within the graphical user interface (GUI).
