Viewing a Model Element's Properties

 Top  Previous  Next

To view the properties of a Model Element within a Model (e.g. an Entity Type), open the Properties Toolbox and click on the Model Element. If the Properties Toolbox is already open on the workbench, simply click on the Model Element.

 

The following steps outline how to view a Model Element's properties within the Properties Toolbox:

 

1.Make sure the Properties Toolbox is open on the Boston workbench
2.Left-click on the Model Element within the Page that you are currently working with within the Boston workbench.
The properties of the Model Element will appear within the Properties Toolbox.
