Role Constraint Conflict Error

 

The Role Constraint listed in the description of the Model Error conflicts with the other Role Constraint listed in the description of the Model Error.

 

Error Resolution

 

To resolve this error, resolve the conflict between the listed Role Constraints.

 

See the chapter: Managing Ring Constraints
